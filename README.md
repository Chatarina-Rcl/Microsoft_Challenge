# Deskripsi Project
Membuat sebuah aplikasi E-Learning sederhana yang berbasis android menggunakan dart. Aplikasi ini meminta user untuk melakukan pendaftaran terlebih dahulu untuk dapat menggunakan aplikasi lebih lanjut. Kemudian setelah login, Home aplikasi akan menampilkan macam-macam mata kuliah yang bisa dipilih oleh user. 

#UGD 2
1. eye toogle : show password pada login page
2. date picker : menambah input tanggal untuk data register
3. gird home : halaman home yang memberikan tampilan grid
4. alert dialog : mengembangkan halaman register
5. checkbox : menambah agreement & policy untuk user
6. floating button : menambah experience user untuk dark mode
